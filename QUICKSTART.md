# ⚡ Quick Start Guide

Get up and running in 5 minutes!

## 🎯 What You Need

1. **Gemini API Key** - Get from https://makersuite.google.com/app/apikey (FREE)
2. **Google Sheet** - For job descriptions database
3. **GitHub Account** - For hosting (FREE)

---

## 🚀 3-Step Setup

### Step 1: Get Gemini API Key (2 minutes)

```
1. Visit: https://makersuite.google.com/app/apikey
2. Click "Create API key"
3. Copy and save the key
```

### Step 2: Create JD Sheet (2 minutes)

```
1. Open Google Sheets: https://sheets.google.com
2. Create new sheet, rename tab to "JD"
3. Add columns:
   A1: Position Name
   B1: Job Description
4. Add your job descriptions
5. Share → Anyone with link can view
6. Copy Sheet ID from URL
```

### Step 3: Deploy to GitHub Pages (1 minute)

```
1. Download this repository
2. Go to: https://github.com/new
3. Create repository named "resume-screener-ai"
4. Upload all files
5. Settings → Pages → Source: main branch
6. Done! Live at: https://YOUR_USERNAME.github.io/resume-screener-ai/
```

---

## 🎨 First Use

### 1. Open Your Site
```
https://YOUR_USERNAME.github.io/resume-screener-ai/
```

### 2. Configure
- Click "Gemini API Configuration"
- Enter API Key
- Enter Sheet ID
- Green light = Ready! ✅

### 3. Use
1. Upload resumes (drag & drop)
2. Select job position
3. Set filters (optional)
4. Click "Analyze"
5. View results!
